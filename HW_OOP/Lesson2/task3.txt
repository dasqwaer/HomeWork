Класс карта
- поле масть;
- поле значение;
- поле положение (рубашкой вверх или вниз).
Класс колода, производный от карта
- поле которое содержит все карты (объекты класса карта) в каком-то определённом порядке;
- метод перетасовать; (хаотично перемещает порядок карт);
- метод раздать. (принимает на вход количество игроков) из поля карт по порядку согласно; каким то правилам (по часовой, по одной, дилеру перевёрнутая) раздаёт карты игрокам.
Класс игрока
- поле игрок/дилер;
- поле набора карт для текущей игры;
- метод сделать ставку;
- метод добрать карту;
- метод перевернуть карту.(только для дилера).
Класс Шафл (игровой процесс)
- принимает количество игроков;
- вызывает метод класса колода для перемешивания перед началом игры;
- Завершает игру и выводит результат.
